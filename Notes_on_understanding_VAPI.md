
Create an Assisstant:
https://docs.vapi.ai/api-reference/assistants/create

What the assisstant initially says is defined by the parameter "firstMessage"
What the assisstant will say and how it behaves afterwards is defined by the parameter "model", where the specific LLM Model to be used in defined and there it can be defined in its own system message.

https://docs.vapi.ai/api-reference/calls/create

To do an outbound Call the number to be called to can be defined inside the object "customer" for "number"
